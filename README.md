## Velkommen til gruppe 7s Mario Projekt

For at vi kan skrive kode sammen har vi valgt at bruge Github. 
Så for at komme i gang skal i...
1. Oprette en Github account
2. Forbinde jeres Github account til jeres computer med SSH key (se guiden [her](src/docs/gitSSHGuide.md))
3. Clone dette repository således...
   1. Find den mappe i gerne vil have projektet i. 
   2. Højreklikke og tryk New terminal at folder (eller tilsvarende i windows)
   3. Kør ```git clone https://github.com/mario-projekt-gruppe-7/mario-projekt-gruppe-7?tab=readme-ov-file```

Se vores git workflow (arbejdsgang) i [gitGuide](src/docs/gitGuide.md)
